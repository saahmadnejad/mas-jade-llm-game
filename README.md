# Adaptive JADE Agents

## Requirements

- Java 8 or higher
- [JADE](https://jade.tilab.com/) framework (add `jade.jar` to classpath)

## How to Run

1. Compile the code:

```bash
javac -cp .:jade.jar src/**/*.java
```

2. Run the app:

```bash
java -cp .:jade.jar ir.donbee.mas.jade.App
```
